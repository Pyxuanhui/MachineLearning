> 包含2000个数据实例

客户细分是将市场细分为具有相似特征的离散客户群。客户细分可以成为识别未满足的客户需求的有力手段。使用上述数据，公司可以通过开发具有独特吸引力的产品和服务来超越竞争对手。

该数据集包含有关客户的信息，如性别、婚姻状况、年龄、教育、收入职业等，我们可以通过这些信息轻松拟合我们的模型以进行更好的预测。
  
您欠一家超市商场，通过会员卡，您可以获得有关客户的一些基本数据，例如客户 ID、年龄、性别、年收入和支出分数。您想了解客户，例如谁是目标客户，以便营销团队了解并相应地规划策略。

### 数据注解
* ID：编号
* Sex：性别
* Marital status：婚姻状况
* Age：年龄
* Education：教育程度
*  Income：收入
* Occupation：占用
* Settlement size：结算规模
